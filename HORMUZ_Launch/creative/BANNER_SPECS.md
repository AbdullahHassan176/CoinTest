# X / Telegram banner — Canva checklist (playbook §5)

**Goal:** One coherent look: logo + strait (map silhouette or photo) + headline optional + **“Not financial advice · DYOR”** in **small** type (readable at thumbnail scale).

**Wrong-coin guard (tiny sub-line):** e.g. *Official $STRAIT — stateofhormuz.org only · not Birdeye $HORMUZ* — same wording as site pin (`app/content/officialStraitMessaging.ts`).

## Suggested export sizes

| Use | Width × height (px) | Notes |
|-----|---------------------|--------|
| X header | 1500 × 500 | Safe text inside center 1200 × 400 |
| Telegram | 1920 × 1080 or 1280 × 720 | TG crops; keep NFA bottom-right |

## Typography

- **NFA line:** ~10–12% of banner height, mono or semi-mono, ~60–70% white opacity — match site `launch-nfa-micro` vibe.
- **Brand:** Gold `#C9A84C` / teal `#00B4CC` from the dApp tailwind theme (`hormuz-gold`, `hormuz-teal`).

## File naming

Export to this folder as `banner-x-tg.png` (or add dated variants `banner-2026-04.png`).
